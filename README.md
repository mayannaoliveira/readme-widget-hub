## ✍ 如何贡献徽章

> \[!Warning]
> 请不要直接修改 [README.md](./README.md) 文件，因为这个文件是通过脚本自动生成的，修改后并不会生效。

1. Fork 本仓库
2. 安装依赖：`npm install`，运行预览 `README.md` 的服务：`npm run dev`（支持热更新）
3. 在 [badge](./badges/) 文件夹下增徽章信息文件，格式参考：[badges/template.json](./badges/template.json)
